SelectBlock ::= ( ( SelectClause ( ( FromClause ( LetClause )? ( WhereClause )? ( GroupbyClause ( LetClause )? ( HavingClause )? )? ) | ( LetClause ( WhereClause )? ) )? ) | ( FromClause ( LetClause )? ( WhereClause )? ( GroupbyClause ( LetClause )? ( HavingClause )? )? SelectClause ) )

FromClause ::= <FROM> FromTerm ( <COMMA> FromTerm )*
